## 一、知识管理核心（7个）

### Dataview `v0.5.68`
用类 SQL 语法从笔记 Frontmatter 和内联字段中提取数据，生成动态列表/表格/日历。最适合构建项目仪表盘、阅读清单、学习进度追踪等需要跨笔记聚合信息的场景。

### Omnisearch `v1.29.3`
全文模糊搜索引擎，支持 PDF 内容索引、中文分词、OCR 图片文字搜索。键盘驱动（`Ctrl+Shift+F`），比 Obsidian 原生搜索快且准。日常找笔记的主力入口。

### DB Folder `v3.5.1` 🆕
把文件夹变成 Notion 式数据库视图——表格/卡片/日历，字段可编辑、可筛选排序。适合不写 Dataview 查询就能管理结构化数据的场景（如客户列表、实验记录）。

### Smart Connections `v4.5.2` 🆕
基于 Embedding 的 AI 语义关联引擎：自动推荐相关笔记、支持 AI 聊天问答（本地或远程模型），生成知识图谱式的可视化链接。适合大量笔记间需要发现隐性关联的研究型工作流。

### Calendar `v1.5.10`
侧边栏日历热力图，点击日期直接打开/创建当日日记。纯视觉层面的每日笔记导航器，与 Periodic Notes 配合使用。

### Daily Notes Editor `v1.1.0`
Roam Research 风格的连续日记编辑视图——不必逐天打开日记文件，一个滚动页面编辑多天的内容。适合日记驱动的写作和回顾。

### Kanban `v2.0.51`
纯 Markdown 驱动的看板，任务以 `- [ ]` 格式存储，列是 `## 列名` 标题。支持日期、标签、按日期自动归档。项目轻量管理首选。

## 二、AI/代码集成（3个）

### MCP Tools `v0.2.27`
把 Obsidian Vault 暴露为 MCP Server，Claude Desktop 可直接读取、搜索、管理你的笔记。支持模板填充、语义搜索、文件批量操作。这是 AI 操作 Vault 的桥梁。

### Claudian `v2.0.18`
在 Obsidian 内嵌 Claude Code、Codex 等编码代理——直接在编辑器里写代码、调试、生成内容，AI 输出直接成为笔记。开发笔记 + AI 协作一体化的核心。

### Terminal `v3.25.0`
在 Obsidian 内打开终端/Shell/Git Bash，支持多 Tab、分屏。开发时不离开 Obsidian 就能执行命令。

## 三、同步/备份（2个）

### Remotely Save `v0.5.25`
笔记双向同步到 S3/Dropbox/OneDrive/WebDAV/COS。增量同步、端到端加密可选。移动端同步的主要方案。

### Obsidian Git `v2.38.3`
自动定时提交 + 推送笔记到 GitHub 私有仓库。提供完整版本历史、diff 对比、变更统计。数据安全的最后防线——任何误删都能恢复。

## 四、写作/编辑（3个）

### Templater `v2.20.5`
高级模板引擎：支持 JavaScript 变量/函数、日期运算、剪贴板内容、文件创建/移动等系统命令。比 Obsidian 原生模板强在可编程性，是自动化工作流的基石。

### QuickAdd `v2.12.3`
全局快捷添加面板（`Ctrl+Q`）：快速捕获想法、执行宏、调用 Templater 模板。支持自定义选择菜单和自动化链。捕获效率的核心入口。

### Advanced Tables `v0.23.2`
增强 Markdown 表格编辑：Tab/Enter 自动对齐、公式计算（类 Excel）、列排序。弥补原生编辑器的表格短板。

### Excalidraw `v2.23.7`

### Style Settings `v1.0.9`
集中配置所有已安装主题/插件的 CSS 变量——颜色、字体、间距、动画。不需要写 CSS 就能深度定制外观。

### Iconize `v2.14.7`
给文件夹和笔记添加 emoji/图标，侧边栏视觉导航增强。对空间记忆友好，尤其文件夹多的时候。

## 六、学术/研究（3个）

### Spaced Repetition `v1.15.1` 🆕
基于 SM-2 算法的间隔复习系统：在笔记中以 `?` 标记问答卡片，根据遗忘曲线自动排期复习。适合考试复习、概念记忆、语言学习等需要长期保持记忆的场景。

### Obsidian Annotator `v0.2.11` 🆕
在 Obsidian 内直接打开 PDF/EPUB 并标注——高亮、区域注释、边注，标注自动同步到对应 Markdown 笔记。填补了 Zotero 集成之外的 PDF 深度阅读需求。

### Paste Image Rename `v1.6.1` 🆕
粘贴图片时自动按规则重命名（如 `YYYY-MM-DD-HHmmss.png`），避免 `Pasted image 20240531.png` 这类无意义文件名。图片管理的基础配置。

## 七、任务管理（1个）

### Tasks `v8.0.0`
跨整个 Vault 的任务追踪引擎：支持到期日、循环、优先级、依赖、自定义状态。通过 Dataview 风格的过滤语法查询聚合任务，如 `not done due before tomorrow`。GTD 方法的落地工具。
