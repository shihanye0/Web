```
# 爬取整站

# 搜索并获取内容

```

**Python SDK**：
```bash

```

```python

# 爬取单页

# 爬取整站

    'limit': 100,
    'scrapeOptions': {'formats': ['markdown']}

```

**REST API**：
```bash
curl -X POST https://api.firecrawl.dev/v1/scrape \
  -H "Authorization: Bearer fc-689e21f8b7f1473fbd42c47291d84af5" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com"}'
```

### 调用方式

**命令行执行**：
```bash

# 激活环境
conda activate E:/conda_envs/scrapling

# 单行命令
E:/conda_envs/scrapling/python.exe -c "from scrapling import Fetcher; print(Fetcher().get('https://httpbin.org/ip').text)"
```

**代码模板**：
```python
from scrapling import Fetcher

# 初始化
fetcher = Fetcher()

# 基础爬取
page = fetcher.get('https://example.com')
print(page.text)  # 页面文本
print(page.html)  # 完整 HTML

# CSS 选择器
title = page.css('title::text').get()
links = page.css('a::attr(href)').getall()
divs = page.css('div.content')

# XPath
elements = page.xpath('//div[@class="name"]/text()')

# 遍历提取
for item in page.css('li.item'):
    name = item.css('span.name::text').get()
    price = item.css('span.price::text').get()
    print(f"{name}: {price}")
```

**批量爬取模板**：
```python
from scrapling import Fetcher
import json
import time

fetcher = Fetcher()
results = []

urls = [
    "https://example.com/page1",
    "https://example.com/page2",
    "https://example.com/page3",
]

for i, url in enumerate(urls):
    try:
        page = fetcher.get(url)
        data = {
            "url": url,
            "title": page.css('title::text').get(),
            "content": page.text[:1000]
        }
        results.append(data)
        print(f"[{i+1}/{len(urls)}] OK: {url}")
        time.sleep(1)  # 礼貌延迟
    except Exception as e:
        print(f"[{i+1}/{len(urls)}] FAIL: {url} - {e}")

with open('results.json', 'w', encoding='utf-8') as f:
    json.dump(results, f, ensure_ascii=False, indent=2)

print(f"完成，共 {len(results)} 条数据")
```

### 特点
- 反指纹检测（浏览器指纹伪装）
- 本地运行，无 API 额度限制
- 支持 Playwright 处理动态页面
- 轻量快速

## 灵活切换决策

### 决策流程图

```
需要爬取网页？
    │
    ├─ 单页快速获取？
    │   └─ → Firecrawl MCP（一行命令）
    │
    ├─ 批量爬取（>10 页）？
    │   └─ → Scrapling（本地运行，无限制）
    │
    ├─ 网站反爬严格？
    │   └─ → Firecrawl（企业级代理池）
    │
    ├─ 需要登录/Cookie？
    │   └─ → Scrapling + Playwright
    │
    ├─ 需要结构化数据提取？
    │   └─ → Firecrawl（内置 AI 提取）
    │
    └─ 需要定时任务？
        └─ → Scrapling（本地 cron 更灵活）
```

### 场景对照表

| 场景 | 首选 | 备选 | 原因 |
|------|------|------|------|
| 快速获取页面内容 | Firecrawl | Scrapling | MCP 直接调用 |
| 批量爬取公司信息 | Scrapling | Firecrawl | 无 API 限制 |
| JS 渲染页面 | Firecrawl | Scrapling | 内置 Playwright |
| 需要代理池 | Firecrawl | - | 云端自带 |
| 需要登录状态 | Scrapling | - | 可保存 Cookie |
| 定时爬取任务 | Scrapling | Firecrawl | 本地 cron |
| PDF/DOCX 解析 | Firecrawl | - | 内置支持 |
