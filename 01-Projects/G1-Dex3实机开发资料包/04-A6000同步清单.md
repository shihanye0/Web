# A6000 同步清单

## 首次同步范围

| 类别 | 同步方式 | 服务器目标建议 | 验收 |
|---|---|---|---|
| 代码、脚本、项目级文档 | Git 优先 | `~/projects/isaac_g1` | `git status` 干净，commit 可追溯 |
| 本资料包 | Git 或 `rsync` | `~/projects/docs/G1-Dex3实机开发资料包` | 四篇 Markdown 均存在 |
| Conda 环境声明 | `environment.yml` / requirements lock | `~/projects/environment/` | 重新创建环境后能 import 核心包 |
| π0.5 基础模型 | 单独 `rsync` | `~/models/pi05_base` | 目录清单与哈希一致 |
| 数据集 | 单独 `rsync` | `~/datasets/<版本名>` | episode/video/meta 完整，统计量可读 |
| 正式 checkpoint | 单独 `rsync` | `~/models/checkpoints/<实验名>` | checkpoint 和 processor 可重载 |
| 官方 SDK/示例 | 从官方源按版本获取 | `~/third_party/unitree_sdk2` | 记录 tag/commit，官方只读样例可运行 |

## 禁止同步

- 密码、token、API key、SSH 私钥、浏览器 Cookie、Codex/Claude Code 登录缓存。
- 无筛选的缓存目录、临时视频、`__pycache__`、重复输出和所有历史 checkpoint。
- 未确认安全边界的实机控制脚本。

## 建议同步顺序

1. 代码和文档。
2. `lerobot312`、`isaaclab`、`g1_hw_sdk` 三个环境的声明文件。
3. 基础模型和一个最小数据集，运行训练/推理 smoke。
4. 完整数据集和正式 checkpoint，执行离线评估。
5. 官方 SDK 和只读 DDS 样例；实机到位后才接入机器人网络。

## A6000 最小验收命令

```bash
nvidia-smi
conda env list
git -C ~/projects/isaac_g1 status --short
```

随后分别在对应环境中验证：π0.5 可以加载基础模型、Isaac Lab 可以启动无界面 smoke、官方 SDK 可以编译或导入。没有通过这些检查前，不传完整数据，不启动正式训练，更不接入真实动作话题。
